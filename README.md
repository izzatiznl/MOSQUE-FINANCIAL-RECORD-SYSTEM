# Mosque Financial Record System

**Mosque Financial Record System** is a web‑based application designed to help mosques manage their financial records efficiently. The system provides structured tracking of income, expenses, budget requests, and automated reporting — with secure role‑based access for administrators, treasurers, and committee members.

---

## 🛠️ Features

### 📊 Financial Management
- **Record Transactions:** Add, view, update, and delete income and expense entries.  
- **Budget Requests:** Submit and approve budget allocations for mosque events.  
- **Automated Reports:** View financial reports summarizing income, expenses, and budgets.

### 👥 User Roles
- **Administrator:** Full access to all system features.  
- **Treasurer:** Manage financial records and reports.  
- **Committee Member:** Access based on assigned permissions.

---

## 🧱 Technologies Used
- **Backend:** Django  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** SQLite (default, can be configured to MySQL)
